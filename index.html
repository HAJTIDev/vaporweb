Features
WHAT'S NEW
IN VAPOR
Vapor helps you organize and launch your games in one place. It keeps things simple, but still gives you useful controls when you need them.
🔍

Smart Detection

Choose a folder and Vapor finds the game files for you. It skips installers, uninstallers, and other files you do not need to launch.

🎨

SteamGridDB Artwork

Vapor can automatically add cover images, banners, and logos so your library looks clean. You can also set your own images anytime.

⏱

Playtime Tracking

Vapor tracks how long you play each game and saves the total time. Your hours are shown directly in your library.

📁

Collections + Favorites

Mark games as favorites and group them into collections. This makes large libraries easier to browse.

🖱

Quick Context Actions

Right-click a game to open quick actions like favorite, collection updates, and file shortcuts.

🎮

Controller + Keyboard

You can use mouse, keyboard, or controller. Move with arrows or D-pad, launch with Enter or A, and go back with Escape or B.

🚀

Auto Updates

Vapor can check for updates and install new versions, so you get fixes and features without manual reinstall steps.

🪟

Start With Windows

Turn this on if you want Vapor to open when Windows starts.

🧲

In-App Torrent Client

Start downloads from magnet links or .torrent files in the app. See progress, speed, and peers, and pause or resume anytime.

🧰

Per-Game Controls

Edit each game's name, executable path, and optional Steam App ID. You can also enable run as admin for games that need it.

📊

Sorting + Filters

Find games faster with search, genre filters, and sorting by name, playtime, last played, or recently added.

🧭

Sidebar Personalization

Change sidebar size, show or hide playtime, and choose sort order so the app feels right for you.

🗂️

Folder + EXE Tools

Open the game folder, reveal the executable file, and manage favorites or collections from one quick menu.

⚙️

Download Control Center

Control all downloads in one place. Set speed limits, clear completed items, and manage active downloads.

🛡

Safer Library Cleanup

Optional confirmation prompts help prevent accidental removals from your library.

How It Works
UP AND RUNNING
IN 3 STEPS

No account setup. Just pick your games folder and start playing.

01

Point at a Folder

Pick the folder where your games are installed. Vapor scans it and finds the files that actually launch your games.

02

Review & Import

Check the list, choose the games you want, and import them. Vapor can also add artwork automatically.

03

Launch & Play

Press Play from your library. Vapor tracks your session and updates playtime when the game closes.

SteamGridDB Integration

ARTWORK THAT SLAPS

Vapor connects to SteamGridDB to add cover art, hero banners, and logos for your games. Your library looks clear and organized instead of plain file names.

  • ▸ Adds cover, hero, and logo images per game
  • ▸ You can paste your own image URL
  • ▸ You can upload local image files
  • ▸ Useful for big games and small indie games

Intelligent Scanning

FINDS THE RIGHT
EXECUTABLE

Game folders often include many extra .exe files. Vapor tries to pick the real game launcher and ignore setup or helper tools.

  • ▸ Skips common non-game files like setup and uninstall
  • ▸ Uses file name and location hints to choose the best match
  • ▸ Avoids duplicate paths during rescans
  • ▸ Keeps scans fast with depth limits

KNOW YOUR
HOURS

When you launch a game from Vapor, it tracks your play session. After you close the game, your total playtime updates automatically.

  • ▸ Saves your last played date automatically
  • ▸ Lets you sort by playtime or recent activity
  • ▸ Optional playtime display in the sidebar
