Get Vapor

Your games in one clean launcher.

Vapor is a fast Windows launcher for DRM-free and store-based PC games. It scans your folders, imports artwork, tracks playtime, and keeps your library organized.

Everything new in the app, in one place

This list reflects current features from Vapor itself, including newer additions like Discord Rich Presence, game folder management, and per-game environment variables.

Smart game scanning

Scans folders for launchable .exe files and avoids installer, uninstall, and helper executables.

Scan all drives

Optional auto-scan checks known game locations across drives when opening Add Games.

Game folder management

Add and manage multiple library folders, each scanned independently.

SteamGridDB artwork

Fetches covers, heroes, logos, icons, and genres to make your library look polished.

Manual artwork override

Set custom art with URL or local file when you want your own look.

Collections and favorites

Create collections, mark favorites, and organize large libraries quickly.

Search, sort, filters

Filter by genre, search by name, and sort by name, playtime, recently added, or last played.

Playtime and recency

Tracks session playtime and last played automatically when games are launched from Vapor.

Sidebar controls

Choose compact sidebar mode, sidebar sort order, and optional playtime display.

Discord Rich Presence

Newer addition: updates Discord status while a tracked Vapor-launched game is running.

Environment variables per game

Newer addition: configure per-game environment variables for mods or launch behavior.

Per-game launch settings

Edit name, executable path, Steam App ID, and run-as-administrator behavior.

In-app torrent downloader

Start downloads with magnet links or .torrent files directly inside Vapor.

Download control center

Pause, resume, remove, clear completed, open folder, and launch setup.exe from downloads.

Global speed limiter

Set a global KB/s speed cap for active torrent downloads plus custom download directory.

Auto updates

Check, download, and install updates with optional auto-check on startup.

Start with Windows

Enable automatic app launch when you sign in to Windows.

Safer library cleanup

Optional confirmation prompts prevent accidental game removal from your library.

How it works

1

Add folders

2

Review and import

Select what to add, then let Vapor fetch artwork and metadata.

3

Play from one library

Launch instantly, track playtime, manage collections, and keep settings per game.
